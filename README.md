# NinjaPhy ⚛️

Site pédagogique de physique-chimie pour les lycéens — cours, TP, exercices et ressources
conformes aux programmes officiels de l'Éducation Nationale.

🌐 **Site en ligne** : [senseininja.github.io/github-slideshow](https://senseininja.github.io/ninjaphy)

---

## 📚 Classes disponibles

| Classe | Programme | Chapitres |
|--------|-----------|-----------|
| **Seconde** | Physique-Chimie (BO 2019) | 12 chapitres – 3 thèmes |
| **Première ES** | Enseignement Scientifique | 10 chapitres – 3 thèmes |
| **Première STI2D** | Physique-Chimie (BO 2019) | 13 chapitres – 4 domaines |
| **Terminale ES** | Enseignement Scientifique | 9 chapitres – 3 thèmes |
